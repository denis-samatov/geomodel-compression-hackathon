# Стартовый набор

Этот каталог содержит минимальное Docker-окружение для локальной проверки решения.

## Назначение

Стартовый набор не должен решать задачу за участника, но должен сокращать время на вход в кейс и снижать число некорректных отправок решения.

## Что здесь есть

| Материал | Зачем это нужно |
|---|---|
| `Dockerfile` | Сборка локального контейнера для воспроизводимой проверки решения |

## Как использовать

1. При необходимости соберите локальный контейнер:

```bash
docker build -t geomodel-judge starter-kit
```

2. Используйте этот образ вместе с [public_evaluator](../public_evaluator/README.md) для локальной проверки решения.
