# Скрипты

Здесь публикуются вспомогательные скрипты для участников.

## Доступные скрипты

| Скрипт | Назначение |
|---|---|
| `pack_submission.py` | Упаковывает решение участника в `tar.gz` |

Публичную модель нужно скачивать напрямую по ссылке из [docs/data-format.md](../docs/data-format.md). В participant-facing версии репозитория здесь остается только утилита упаковки решения.
