# Скрипты

Здесь публикуются вспомогательные скрипты для участников.

## Доступные скрипты

| Скрипт | Назначение |
|---|---|
| `pack_submission.py` | Упаковывает решение участника в `tar.gz` |
| `sync_github_labels.py` | Проверяет и синхронизирует метки из `.github/labels.json` |
| `validate_github_meta.py` | Проверяет целостность `.github/`: форм обращений, меток, автоматических процессов и зон ответственности |

Публичную модель нужно скачивать напрямую по ссылке из [docs/data-format.md](../docs/data-format.md). В репозитории остаются только служебные скрипты для упаковки решения и сопровождения инфраструктуры.
