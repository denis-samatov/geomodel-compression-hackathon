# Таблица результатов

Этот каталог содержит публичную таблицу результатов и правила ее публикации.

## Публикуемые артефакты

| Файл | Назначение |
|---|---|
| `current.csv` | Текущий публичный снимок таблицы результатов |
| `current.json` | Версия в формате JSON для GitHub Pages или внешней витрины |
| `history.csv` | Опциональная история обновлений по расписанию |
| `update_leaderboard.py` | Нормализует CSV и синхронизирует `current.json` |

## Что не публикуется

- приватная оценка;
- скрытые датасеты;
- сырые журналы системы оценки;
- подробные трассировки ошибок;
- скрытая детализация метрик.

## Частота обновлений

- Рейтинговая таблица команд обновляется один раз в день вне зависимости от количества коммитов решений

## Структура таблицы
| Файл | Назначение |
|---|---|
| `rank` | Место команды в общем рейтинге на момент последнего обновления
| `team_name` | Команда | |
| `best_public_score` | Суммарная оценка решения команды по трём метрикам |
| `compression_ratio` | Процент результирующего сжатия данных |
| `runtime_bucket` | Время работы алгоритма сжатия и восстановления |
| `last_valid_submission_at` | Последняя публикация решения |
